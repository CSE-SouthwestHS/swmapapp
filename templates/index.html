How it was made?

The foundation of this perfectly optimized web app is based on a simple HTML frame and neatly decaorated with highly complex Cascading Style Sheets (CSS). Javascript is used for most of the functionality including buttons, toggles, and displaying navigational features. The actual navigational functionality, however, is reserved to the Python code. With a complete index of hallways and rooms converted to individual nodes, the Python code finds the shortest route between the nodes, providing the user with a quick, reliable navigation app.
